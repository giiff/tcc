\section{O que devo escrever aqui?}%

Cada ferramenta, tecnologia, definições e todo o arcabouço teórico do trabalho deve estar em seu \refCap{Referencial_Teorico}.
Aqui, no \refCap{Metodo} você deve escrever como utilizou as ferramentas, tecnologias e outros recursos para resolver o problema proposto com vistas a alcançar seus objetivos.
Este não é lugar para definir nada novo.


Não caia na tentação de dizer aqui os resultados! Guarde-os para o \refCap{Resultados}. Muitas vezes é interessante começar a escrita, justamente pelo \refCap{Resultados}.
